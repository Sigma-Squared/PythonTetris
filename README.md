# PythonTetris
Tetris with Python &amp; Pygame

---

## To run (requires [PyGame](http://www.pygame.org/hifi.html))
run tetris.py in python2
Unix-like:
```
python tetris.py
```
Windows:
```
py -2 tetris.py
```

---
## Controls
+ __UP__ Rotate Block
+ __DOWN__ Accelerate Block
+ __LEFT__ Move Block Left
+ __RIGHT__ Move Block Right
